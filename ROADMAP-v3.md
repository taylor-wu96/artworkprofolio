# ROADMAP v3 — 從「一個連續的身體」到「一個有體溫、有記憶的身體」

> **產品迭代藍圖 v3**（2026-06-28）
> 配套：[ROADMAP.md](ROADMAP.md)（v1・瘦身/通電/擴音）／[ROADMAP-v2.md](ROADMAP-v2.md)（一體性＋護欄）／[DESIGN.md](DESIGN.md)（視覺設計）／[PROGRESS-REDESIGN.md](PROGRESS-REDESIGN.md)（工程落地）／[`.agents/skills/Design.md`](.agents/skills/Design.md)（母文件通則）

## 〇、為什麼有 v3

[v1](ROADMAP.md) 造齊儀器（放映 Grid、真 EXIF、OG 立面、論述面板、Series 連續放映）。
[v2](ROADMAP-v2.md) 把儀器**連成一個會連續移動的身體**（顯影即移動的跨頁過場、連續的暗、三道門指路、空房尊嚴），並立起護欄（Playwright 視覺回歸、`lint:design` 綠契約、`onPage` 腳本生命週期、DoD、E6 VT 法律）。

但讀完當前原始碼後，有三個誠實判斷讓 v3 必須存在：

### 0.1 v2 的一體性是「空間連續」，還沒有「時間連續」（感性）

你在一片黑裡移動不中斷了——這是**空間**的一體性。但這個身體還沒有**時間**的一體性：

| 斷點 | 證據 | 體感 |
|---|---|---|
| **「活」是一堆各跳各的閃爍，不是一個心跳** | `wip-breathe` 3.6s（[components.css](src/styles/layers/components.css)）／`sound-wave` 1.4s（[layout.css](src/styles/layers/layout.css)）／3D 綠光 `sin(t·0.12)`≈52s／audio LFO 20s（[audio.ts](src/scripts/audio.ts)）——每個有自己的時鐘 | 館在很多處閃，卻不像「一個東西在呼吸」 |
| **光只活在 hero，不持續** | [Hero3D.jsx](src/components/Hero3D.jsx) 的光場捲過一屏就隨 `.hero`（`100svh`）一起捲走；圖錄頁背後是純黑 | DESIGN §9 第二樂章「稀釋成圖錄背後的環境殘光」是宣言，工程沒兌現 |
| **衰敗只活在 3D，不在 2D 檔案裡被感覺到** | 侵蝕曲面承載衰敗；但索引裡時間只是 `year-marker` 一行字 | 「衰敗是時間的誠實」沒被身體感覺到，只是資訊 |

### 0.2 器官建好了，但沒通血（理性／產品）

[DESIGN §8.2](DESIGN.md)、[v2 E5/H3](ROADMAP-v2.md) 都親口說「這是當前**唯一的真風險**」卻都推給「內容節奏」當尾巴：真 EXIF 顯示面、Series 連續放映、themes facet——**全部建好但休眠**（種子圖無 EXIF、資料集無 series 文件）。一座**半亮的館比全暗更糟**：它讓人以為有人照顧、其實器官在空轉。v3 把通血升為**一級工作**——否則接下來所有感官精修，都是在一個**沒有內臟的身體**上做表面。

### 0.3 對熟人直覺，對陌生人還薄（理性／UIUX）

這個身體對你自己（做它的人）很直覺，對第一次來的觀者還薄：
- **第一次接觸**：陌生人落在一片暗的 3D hero（[index.astro](src/pages/index.astro)），沒有「往下捲」的暗示、沒有一句空間定位。
- **三道門被「讀到」了但還沒被「感覺到」**：v2 的策展短語＋[RoomNav](src/components/RoomNav.astro) 在**資訊層**釐清了，但**體感層**——系列索引仍退回 `.archive` 與作品同形。
- **尋路只有一條無盡的時間捲軸**：作品到 30、50 件時，純倒敘＋themes facet 會塌。

> **v3 的脊椎**：v2 讓身體**連續地移動**；v3 讓它**有體溫（一致的呼吸）、有光的記憶（持續的殘光）、有年歲（衰敗被時間刻出）**，並且**真的通血**（內容點亮器官），讓陌生人能憑本能走進來。
> 工程仍是手段，審美仍是目的。這一輪工程的角色是**節拍器與輸血管**：讓全館的「活」對齊同一個心跳，讓建好的器官真的供血。
>
> **給未來的自己**：當你又開始把 C1 通血當「之後再說的內容工作」、把 T 系列當「正事」時，停下來。順序剛好相反——**一個沒有血的身體，再精緻的呼吸也是裝的。**

## 一、兩條線

```
甲線・一體性的下一層（感性）── 從空間連續 → 時間與材質的連續
   T1 一致的呼吸（One Breath）       T2 持續的光（Persistent Light）
   T3 門檻作為儀式（Threshold Felt）  T4 衰敗刻出年歲（Decay as Age・實驗）

乙線・通血與直覺的誠實審計（理性）── 讓器官供血、讓陌生人能憑本能走
   C1 通血：讓休眠器官亮起    C2 第一次接觸    C3 三道門的差異被感覺到    C4 尋路審計
```

**核心排序**（延續 v2 紀律）：**乙線的 C1 通血必須先於甲線的感官精修。** 沒有內臟，不在身體上做表面。

## 二、三個地平線

```
H0 先通血・先反思 ── 不動感官，先解掉「空殼冒充完成」、先把直覺斷點審計清楚   （本週・解鎖一切）
   │   C1 通血（一條真內容走通整條顯示面）   C2/C3/C4 審計（產出判斷）
   ▼
H1 一體性的下一層 ── 通血後才在有內臟的身體上做時間/材質的連續              （最有感）
   │   T1 一致的呼吸    T2 持續的光    T3 門檻儀式
   ▼
H2 候選・實驗 ── 高風險或需內容量到門檻才做                                （ROI 逐項驗）
       T4 衰敗刻齡（風險於作品純粹性）   C4 尋路（內容量達門檻才造）
```

# H0 ｜先通血・先反思

## C1 — 通血：讓休眠的器官亮起（最高優先・產品決斷，非功能）

### 🎨 藝術／設計目標
一座館的可信度，不在儀器多精良，在它**真的活著**。EXIF 讀檔、Series 放映、theme 凝視——這些是館的**內臟**，目前在空轉。讓**至少一條真內容走通整條顯示面**，是把「建好但休眠」從冒充「完成」裡揪出來。一座會衰敗的美術館，第一口氣得是真的。

### 🧭 UIUX 細節（最小可信血量）
- 一批**帶真 EXIF 的真照片**：點亮 [signature.js](src/lib/signature.js) `readExif`、Plate `data-exif`、放映 `.pjx__exif`、作品頁 `work__exif` 的「讀取檔案」揭示。
- 一條**真 series**：點亮 Series 連續放映、讓「系列」這扇門不再是空房。
- 一個**真 theme**：走通凝視頁。
- **誠實 IA 為底線**：顯示面無真內容時，該門依 G3 隱藏（`NAV_COUNTS`／`RoomNav` 已就緒），**而不是展示空殼**。

### 🔧 工程：根因／手段／驗收
- **根因**：種子圖皆為無 EXIF 截圖／生成圖（見 `exif-read-surfaces` 記憶）；資料集無 series 文件。
- **手段**：gallery 圖 metadata 抽取需**重部署 Studio**（沿用 `schema-deploy-crash-workaround`：manifest extract ＋ `--no-extract-manifest` 繞 exit 134）＋**重傳回填**（既有影像不回填）。建立 1 條 series 文件、1 個 theme 掛真作品。
- **驗收（把 v2 緩議的 E5 做硬）**：每個顯示面至少一筆真內容在跑，否則該門隱藏。**這是內容＋infra 工作，需真照片——是創作者的手，工程只負責讓門誠實開合。**
- **依賴**：無。**先於一切。**

## C2／C3／C4 — 直覺的誠實審計（產出判斷）

把「是否足夠直覺」做成判斷，動手與否交給 H1/H2：

- **C2 第一次接觸**：審計陌生人落在暗 hero 的前 5 秒——有沒有「往下捲」的暗示、一句空間定位、3D 在手機/弱機/`reduced-motion` 下是否仍是個「邀請」。→ **產出已落地**：見 [§H1·C2](#c2--第一次接觸入口的邀請)。
- **C3 三道門的差異**：作品（單欄敬畏）／影像（印樣密排）／系列（**退回 `.archive` 同形**）的視覺語言分化夠不夠。→ **產出已落地**：系列獨立形態，見 [§H1·C3](#c3--三道門的差異被感覺到系列的獨立形態)。
- **C4 尋路**：作品達 30–50 件時純時間捲軸＋themes facet 會不會塌。→ **產出（待動手）**：索引內 facet 篩選，或解鎖 🔒 HOLD 灰階地圖（真 GPS≥15）。內容量沒到門檻不造。

# H1 ｜一體性的下一層：時間與材質的連續

> 通血就位，現在在一個有內臟的身體上，做 v2 沒碰的那層——把「活」收成一個心跳，把光變成有記憶的材質，把跨越變成被感覺的儀式。

## T1 — 一致的呼吸（One Breath・最低成本最高一體性）

### 🎨 藝術／設計目標
這座館的「活」現在是**一堆各跳各的閃爍**——綠脈動、WIP 呼吸、CV 節點、聲音、3D 綠光，每個有自己的時鐘。一個生命體不該有五個心跳。讓所有活訊號**對齊同一個母拍**，整座館才從「多處在閃」變成「**一個東西在呼吸**」。這是把 DESIGN §4.1「綠＝整座館唯一的活點」從「一粒」升級為「一次共同的吸吐」。

### 🧭 UIUX 細節
- 定母拍 `--pulse: 3.6s`（館的心跳；取自既有 `wip-breathe` 的 3.6s，最沉靜的那個）。
- 所有循環「呼吸」是它的諧波或同源：WIP 綠標＝母拍；聲音波形＝母拍／3 的快諧波；3D 綠光的脈動週期＝母拍的整數倍封套（同源、相位鎖）。
- **跨介質誠實**：CSS（DOM）能共用 `var(--pulse)`；WebGL（3D）與 WebAudio 讀不到 CSS 變數，故以**同一個數字 `3.6` 為基底推導頻率**並寫進註解——它們是「諧波鎖定」，不是「碰巧接近」。
- `reduced-motion`：全部凍結為靜態（延續 v2 合規）。

### 🔧 工程：動哪些檔／驗收
- [tokens.css](src/styles/layers/tokens.css)：`--pulse` 母拍進地基④（由 `lint:design` 的時長規則守住，禁新增裸時長方言）。
- [components.css](src/styles/layers/components.css) `wip-breathe`、[layout.css](src/styles/layers/layout.css) `sound-wave` 改吃 `var(--pulse)` 諧波。
- [Hero3D.jsx](src/components/Hero3D.jsx)：綠光脈動頻率自 `PULSE=3.6` 推導（諧波鎖）。
- **驗收**：視覺回歸（動畫在快照中 disabled，不增 flaky）；新動態不得引入裸時長。
- **依賴**：護欄。

## T2 — 持續的光（Persistent Light・最有感的一手）

### 🎨 藝術／設計目標
兌現 DESIGN §9 第二樂章。光場不在 hero 結束就死，而**稀釋成圖錄背後極淡的環境殘光**。這是「一整座館共用同一片黑」從**宣言變事實**的最大一手——黑不再只是底色，**黑裡留著光走過的記憶**。[global.css](src/styles/global.css) 的「無漸層」鐵律明文「**光場除外**」——這道殘光正是光場的殘餘，是被允許的那個例外。

### 🧭 UIUX 細節
- 暗房（且僅暗房）的內容背後，墊一層**極淡、頂部錨定、向下淡出**的殘光暈——像 hero 的光場沉降後留在牆上的餘溫。
- **克制到接近潛意識**：用既有 `--gallery-ink` 經 `color-mix` 極低比例融入 `--gallery`，**不新增任何色號**；綠仍只在那一粒活光點，殘光是無彩的暖灰光，不帶綠。
- 影像的存在感永遠高於它（DESIGN §3 層 2）。

### 🔧 工程：動哪些檔／驗收
- **極低成本**：不讓 3D 持續高頻跑（耗電）。改用一層 `position:fixed` 的 CSS 殘光（`html[data-room='dark'] body::before`），純靜態漸層，零 JS、零 GPU 負擔——承接 [works-runtime](src/scripts/sketches.ts) 離屏釋放的精神。
- 動 [base.css](src/styles/layers/base.css)（或 layout 層）。`lint:design` 不得新增裸 hex。
- **驗收**：暗房各頁型視覺回歸更新 baseline（這是刻意改版，跑 `test:visual:update`）；亮房不受影響。
- **依賴**：T1（殘光的極慢起伏吃母拍）。

## T3 — 門檻作為被感覺到的儀式（The Threshold, Felt）

### 🎨 藝術／設計目標
站名就是「**門檻**」。但跨房／進作品那一刻，目前是 [room-veil.ts](src/scripts/room-veil.ts) 的功能性淡出——概念脊椎裡最核心的字，還沒被身體感覺到。讓「跨越」成為一個**被感覺**的手勢：一道光的閾值掃過的瞬間。把概念變成可被身體經驗的事件（Turrell 的光閘）。

### 🧭 UIUX 細節
- 與 v2 的 room-veil（跨房）整合而非疊加——同房仍是顯影即移動（G4），跨房（暗↔亮）那一下，veil 淡出時多一道**水平發光門檻線**掃過（光從中央漫開的閾值）。
- **不新增動態方言**：全走既有 `--dur`／`--ease` token。
- `reduced-motion`：退回 v2 既有的瞬切直給（room-veil 已 `if (reduce) return`），門檻線不出現。

### 🔧 工程：動哪些檔／驗收
- **純加性**：只在 [layout.css](src/styles/layers/layout.css) 的 `#room-veil` 上加一個 `::after` 門檻線，**不動 [room-veil.ts](src/scripts/room-veil.ts) 的 JS 時序**（避免碰 v2 警告的「最脆弱的全域過場」）。
- **驗收**：跨房導覽（首頁/作品 → 散文/關於）親眼看門檻；同房導覽不受影響；快照不捕捉過場中間態（安全）。
- **依賴**：護欄、DoD 三條（改全域過場，最危險）。

# H2 ｜候選・實驗（預設懷疑・ROI 逐項驗）

## T4 — 衰敗刻出年歲（Decay as Age・高風險實驗）

### 🎨 藝術／設計目標
「衰敗是時間的誠實」目前只活在 3D 侵蝕面。讓它在 2D 檔案裡也被感覺到——越舊的作品在索引裡越「沉」一點點。時間性從 `year-marker` 的**資訊**，升為可被感覺的**物質狀態**。

### ⚠️ 鐵律與風險
- 直撞 DESIGN §5「**影像不裁切、不污染作品**」。故鐵律是——**衰敗只作用於 chrome（編號、資料條、髮絲線的年歲感），絕不碰影像本身**。
- 極度克制、可逆。先 spike 驗證「美還是髒」，醜就砍。

### 🔧 工程
- 索引條目依年代差，對 `.entry__meta` 一族（**非** `.entry__img`）施極微的去飽和／降透明。純 CSS 自訂屬性驅動，無 JS。標**實驗**，可一行回退。

## C4 — 尋路（內容量到門檻才造）
- 依審計：索引內 facet 篩選，或解鎖灰階地圖（維持 🔒 HOLD，真 GPS≥15）。**內容量沒到門檻不造**——否則是功能劇場。

---

## 三、優先序（若只能做三件）

1. **C1 通血** — 沒有它，所有一體性都是在沒內臟的身體上做表面。**最先。**（需真照片，創作者的手）
2. **T1 一致的呼吸** — 一個母拍把分散的「活」收成一個心跳。一體性 ROI 最高、風險最低、最便宜。
3. **T2 持續的光** — 把「共用一片黑」從宣言變事實，最有感的感官一手。

---

## 四、給每一次改動的 DoD（延續 v2 七問 ＋ v3 第八問）

沿用 v2 的四問（藝術）＋ 三條（兩房／手機／導覽過去）。**新增第八問，專治動態方言增生**：

> **8. 這個改動讓「活訊號」更像同一個生命體的同一次呼吸，還是多了一個各跳各的閃爍？**（任何新動態必須吃 T1 母拍 `--pulse`，且綠仍只在白名單）

護欄延續：任何 T 系列動全域過場／動畫，**必跑 `npm run test:visual` ＋ `npm run lint:design`**；母拍進 tokens 由 lint 守；每條 T 都要有 `reduced-motion` 靜態退回。

## 五、決策紀錄

| 日期 | 版本 | 關鍵決策 |
|---|---|---|
| 2026-06-28 | v3 | 建立 v3。脊椎＝從「連續移動的身體」到「有體溫（一致呼吸）、有光的記憶（持續殘光）、有年歲（衰敗刻齡）、且真的通血」的身體。核心洞見：v2 的一體性是**空間連續**，v3 補**時間與材質連續**；且把被推兩輪的「通血」升為一級工作（C1 先於一切）。T1 立 `--pulse` 母拍統一全館呼吸（跨介質以同一數字諧波鎖）；T2 殘光以「光場除外」條款合法落地、純 CSS 零 GPU；T3 門檻純加性、不動 veil JS 時序；T4 衰敗刻齡標實驗、只碰 chrome 不碰影像。新增 DoD 第八問治動態方言增生。 |
