## 〇、命題：缺的不是哲學，是「執行的精緻度追上哲學」

把七個參照站通讀後，設計語彙收斂成六條，而本站**已體現其中五條**：影像至上、留白即節奏、
克制即奢華、viewing-room 式序列下卷、影像的顯影。唯二未達標的，都在**執行層**：

1. **系統一致性**（參照站最強、本站最弱）——標題尺度七套並存、留白靠散落手寫 `clamp()`、
   CJK 標題在字重 300 偏脆。參照站的「composed」感來自一致的呼吸；本站是每段各喘各的。
2. **被感覺到的觀看**——揭示是「開關」不是「編排」；準星偏吵；放映開場是純 fade；
   影像對 hover 無回應。

> **世界觀（延續 v4/v5，且被創作者半年來每一刀證實）**：這半年的每個動作都是**減**——
> 移除大氣層、殘光、裝訂線、各處邊框，把明體換成冷冽黑體。v7 完全站在這條線上：
> **質地來自照片本身、來自字與留白的工藝，不來自疊在黑上的網頁特效。**
> 站名「潮濕的質地」承諾的濕氣，由影像與工藝兌現，不由 CSS 假材質兌現。
>
> **鐵律**：任何一項都先問 v5 護欄——「這是減法/校準，還是新儀器？」新儀器預設拒絕。
> v7 全部是校準與減法，只有一個例外（B4），且它是把既有語言講完整、非新增。

## 一、三條線（延續感性／理性雙體）

### 甲線 · 材質即精度 —— 最高槓桿、最低風險、全站受惠（理性偏重）

> 感性：一個有把握的字級，比七個猶豫的字級更高級。參照站的貴氣，一半來自字的確定性。

- **A1 標題尺度收成一套** — 現況七套並存：`--display` `clamp(1.2,3.6vw,1.6rem)`
  [tokens.css:77] 卻被各面覆寫——區段標題 `1.7–2.3/300`、作品標題 `1.6–2.1/300`、
  影像集壓圖標題 `1.55–2.4/400`、散文標題 `1.6–2.2/600`、about 名 `1.8–2.8/600`、
  圖錄條目 `1.5625/400`。**收成兩檔確定 token**：
  `--title-display`（立面大標）＋ `--title-work`（單件題名），全立面吃 token、消滅覆寫。
  字重維持房間驅動（暗房展覽輕量／亮房閱讀半粗），只統一「尺度」。
- **A2 CJK 標題排版校準** — 思源黑體 300 在大字發虛。標題設 CJK 專屬 `letter-spacing`
  與 `line-height`，`text-wrap:balance` 推廣，斜線混排（「索引 / Works」）節奏重配。
- **A3 留白節奏收進 token** — DESIGN §8.2 自承散落手寫 `clamp()`。立 2–3 個節奏 token
  （`--space-section` / `--space-block` / `--page-top` 已有），統一 section-head/entry/
  work/plates 的垂直呼吸；可被 `lint:design` 盤點。
- **A4 metadata 降噪**（承 v5 S1/S3）— 首頁 [entry__meta](src/styles/layers/components.css:25)
  一次擠 num/date/themes/sig/latest/wip，比參照站「安靜標牌」吵。重排層級：影像＋標題為主，
  資料退成 whisper（更小、更淡、次序後移）。

### 乙線 · 被感覺到的觀看 —— 感性偏重，以減法為主

- **B1 揭示編排** — [data-reveal](src/styles/layers/media.css:39) 單一 opacity+translateY
  一次到位。精修 easing/距離，條目內加極輕 stagger（影像先落定、圖說一拍後跟上）。
  吃既有 `--dur-slow`/`--pulse`，不新增動態方言（DoD 8）。
- **B2 準星收成低語** — [reticle](src/styles/layers/machine.css:42) opacity 0.92、
  十字＋四角括弧、lerp 0.25。往「更少」精修：平時近乎隱形，只在 lock 作品時四角括弧
  安靜收框；阻尼調更重（Turrell 沉思）。順帶減輕 v4 §0.3「桌機限定精緻」的失衡。
- **B3 影像的觸覺回應** — 印樣格 hover 只有標題轉綠 [gallery/index.astro:165]，影像不動。
  加一絲「顯影」式亮度/對比微升（非網頁式放大），reduced-motion 靜止。
- **B4 放映的抬升 ★唯一的「加」（延續顯影即移動）** — 點 zoomable 進
  [放映](src/styles/layers/media.css:66)目前是純 opacity fade。用 shared-element/FLIP
  讓被點影像「從頁面抬升進放映層」，把既有 G4 `transitionName`「顯影即移動」語言
  貫徹到放映開場。可一行回退成 fade；reduced-motion 退回 fade。

### 丙線 · 版面的策展溫度

- **C1 影像集轉編輯式不規則尺度 ★已拍板** — 現況統一 3 欄 3:2 密排。改為
  一張 hero（用 `featured` 指定）＋其餘收小的 staggered/editorial 節奏，讓尺寸承載
  策展重音（對標 viewing-room / Hauser & Wirth）。**必須下沉全域 components 層**，
  不放頁級 `<style>`（VT 塌陷法律，見 [vt-grid-cjk-collapse] 記憶）。
- **C2 hero 編輯性** — 3D 沉思內核保留；精修落地文字（title/index/sub）構圖與尺度，
  更像參照站「大而克制的編輯字」。hero title 刻意不併入 A1 共用 token
  （它疊在 3D 上、有 mix-blend，需獨立節制）。

## 三、每一次改動的 DoD（延續既有十問，再加一條）

> **11. 這個精修，是把已宣稱的哲學做到位，還是在哲學之外加東西？**（v7：工藝而非儀器）
>
> 重申：v5 第十問（賺得的 vs 無差別套上）、v4 第九問（顯示的是真的嗎）、
> v3 第八問（新動態吃 `--pulse`、綠只在白名單）、v2 三條（兩房/手機/導覽過去）。
> 護欄延續：動全域過場/動畫必跑 `npm run test:visual:update`（v7 是刻意改版，重生 baseline）
> ＋ `npm run lint:design`（綠契約不破）；每條改動有 `reduced-motion` 靜態退回；
> 減法也要證據——先在一頁 spike 驗「這樣更好」再全站施行；每一項能一行回退。
